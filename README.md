# 15 Days of Verilog 🚀

A beginner-to-intermediate Verilog learning journey with daily projects, simulations, and digital design concepts.

---

## 📌 About This Repository

This repository documents my **15-day Verilog learning roadmap** focused on:

* Digital Logic Design
* Verilog HDL
* RTL Design
* Testbenches
* Simulation
* FPGA Fundamentals

Each day includes:

* Verilog design files
* Testbenches
* Simulation outputs
* Small practical projects
* README explanations

---

# 🛠 Tools Used

* Icarus Verilog
* GTKWave
* VS Code
* Git & GitHub

# 📅 Learning Roadmap

| Day | Topic             | Project                  |
| --- | ----------------- | ------------------------ |
| 1   | Logic Gates       | AND, OR, XOR Gates       |
| 2   | Adders            | Half Adder, Full Adder   |
| 3   | Multiplexers      | 2:1 & 4:1 MUX            |
| 4   | Decoder & Encoder | 2-to-4 Decoder           |
| 5   | ALU               | 4-bit ALU                |
| 6   | Flip-Flops        | D Flip-Flop              |
| 7   | Counters          | 4-bit Counter            |
| 8   | Shift Registers   | SISO, SIPO               |
| 9   | FSM               | Traffic Light Controller |
| 10  | Sequence Detector | Binary Pattern Detector  |
| 11  | UART TX           | UART Transmitter         |
| 12  | UART RX           | UART Receiver            |
| 13  | Memory            | RAM Module               |
| 14  | SPI               | SPI Master               |
| 15  | Final Project     | Digital System Project   |
