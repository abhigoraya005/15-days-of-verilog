# Day 14 – SPI Master (Serial Peripheral Interface)

## 🎯 Objective

Design and simulate a **SPI (Serial Peripheral Interface) Master** using Verilog HDL.

This project demonstrates how a master device communicates with SPI slave devices by transmitting serial data using the SPI protocol.

---

# 📖 Introduction

SPI (Serial Peripheral Interface) is a high-speed synchronous serial communication protocol widely used in embedded systems and FPGA designs.

Unlike UART, SPI uses a shared clock generated by the master and supports full-duplex communication between one master and one or more slave devices.

In this project, a simplified **SPI Master** is implemented to transmit 8-bit data serially.

# 🏗 SPI Architecture

```
                 +----------------------+
                 |      SPI Master      |
                 |                      |
 Clock --------->|                      |
 Start --------->|                      |
 Data In ------->|                      |
                 |                      |
                 | MOSI -----------> Slave
                 | MISO <----------- Slave
                 | SCLK -----------> Slave
                 | CS -------------> Slave
                 +----------------------+
```

---

# 📚 SPI Signals

| Signal | Description |
|---------|-------------|
| MOSI | Master Out Slave In |
| MISO | Master In Slave Out |
| SCLK | Serial Clock |
| CS | Chip Select |

---

# 📡 SPI Communication Frame

```
CS

0 --------------------------------------------- 1

SCLK

_|‾|_|‾|_|‾|_|‾|_|‾|_|‾|_|‾|_|‾|

MOSI

D7 D6 D5 D4 D3 D2 D1 D0
```

The SPI Master transmits one bit of data on each clock cycle while the Chip Select (CS) line remains active.

---

# ⚙ Working Principle

The SPI Master performs the following sequence:

1. Waits for the **Start** signal.
2. Pulls the **Chip Select (CS)** line LOW.
3. Loads the input data into a shift register.
4. Generates the SPI Clock (SCLK).
5. Sends one bit at a time through the MOSI line.
6. After transmitting all 8 bits, releases the Chip Select line.
7. Transmission completes.
