# Day 5 – 4-bit Arithmetic Logic Unit (ALU)

## Objective

Design and simulate a 4-bit Arithmetic Logic Unit (ALU) capable of performing arithmetic and logical operations.

---

## Operations Implemented

| Select | Operation   |
| ------ | ----------- |
| 000    | Addition    |
| 001    | Subtraction |
| 010    | AND         |
| 011    | OR          |
| 100    | XOR         |
| 101    | NOT         |
| 110    | Increment   |
| 111    | Decrement   |
